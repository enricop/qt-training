Compass
=======

Create a compass widget. The widget draws a circular compass with needle
capable of displaying 360 degrees. The direction of the compass can be set
via slot methods, and a signal is emitted when the direction changes. A
minimal stubbed out version of the widget is provided as a starting point.

Task
----

* Decide on appropriate programming interface for widget
* Implement the compass widget
* Connect the compass widget to the QSpinBox provided in the lab

Optional
--------

* Set the direction using mouse clicks
